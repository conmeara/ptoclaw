# Terms

PTOClaw is provided as open-source software under the MIT License.

It is a planning aid, not an authoritative payroll, HR, employment, or legal system. Users should verify PTO balances, accrual rules, holidays, blackout dates, and approved leave with their employer's official systems.

Calendar sync is dry-run only in this release. Any future calendar-writing adapter should be reviewed and tested against the user's own calendar setup before relying on it.
